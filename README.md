## 2. OpenAPI Visualization

🔎 [Open the published Swagger UI](https://manuelalfarosierra-cpu.github.io/osapiens-backend-challenge/)

This link opens the published Swagger UI hosted with GitHub Pages. It mirrors the interactive OpenAPI view served by the application and includes the documented endpoints, request examples, response schemas, and the generated API specification used by the project.

When the server is running, the API documentation can also be accessed locally at:
- `http://localhost:3000/docs/swagger` for Swagger UI
- `http://localhost:3000/docs` for ReDoc
- `http://localhost:3000/openapi.json` for the OpenAPI JSON specification
- `http://localhost:3000/openapi.yaml` for the OpenAPI YAML specification
